  <action>
Create directory `platform/src/components/` if it does not exist. Create `NotificationBell.tsx` as a "use client" component.

**Structure:**
```
"use client";
imports: useState, useEffect, useRef from react; createClient from @/lib/supabase/client; Notification type from @/lib/types; Bell, CreditCard, CheckSquare, Plus from lucide-react; formatDistanceToNow from date-fns; es locale from date-fns/locale; useLocale, useTranslations from next-intl; useRouter from next/navigation
```

**Component: `export function NotificationBell()`**

No required props -- the component self-manages auth and data (per NOTIF-09, must mount without passing user context).

**State:**
- `notifications: Notification[]` -- initial empty array
- `open: boolean` -- dropdown visibility
- `pulse: boolean` -- badge animation flag
- `userId: string | null` -- resolved from supabase.auth.getUser()
- `bellRef: useRef<HTMLDivElement>(null)` -- for click-outside detection

**`createClient()` call:** At component scope (NOT inside useEffect), matching portal/layout.tsx pattern.

**useEffect #1 -- Init + Realtime (deps: []):**
1. Call `supabase.auth.getUser()` to get authenticated user ID. If no user, return early (no bell for unauthenticated).
2. Set userId state.
3. Fetch initial notifications: `supabase.from("notifications").select("*").eq("user_id", userId).order("created_at", { ascending: false }).limit(20)`. Set state with result.
4. Subscribe to Realtime channel:
   - Channel name: `"notifications:" + userId`
   - Event: `postgres_changes`, event: "INSERT", schema: "public", table: "notifications", filter: `user_id=eq.${userId}` (per NOTIF-04, defense-in-depth cross-tenant filter)
   - Handler: prepend `payload.new as Notification` to state array, set `pulse` to true, `setTimeout(() => setPulse(false), 2000)` (per D-09)
5. Return cleanup: `supabase.removeChannel(channel)` (critical for React Strict Mode double-mount, per research pitfall #2)

**useEffect #2 -- Click-outside (deps: [open]):**
Per D-07: When `open` is true, add `mousedown` listener on `document`. If click target is outside `bellRef.current`, call `setOpen(false)`. Clean up listener on unmount/close.

**useEffect #3 -- Escape key (deps: []):**
Per D-07: Add `keydown` listener. If `e.key === "Escape"`, call `setOpen(false)`. Clean up on unmount.

**Computed values:**
- `unreadCount`: `notifications.filter(n => !n.read).length`
- `badgeLabel`: `unreadCount > 9 ? "9+" : String(unreadCount)` (per D-01)

**Functions:**

`markRead(id: string)`:
- `supabase.from("notifications").update({ read: true }).eq("id", id)`
- Update local state: map over notifications, set matching id to `read: true`
Per D-05: Also navigate to `action_link` if present using `router.push(action_link)`. Close dropdown.

`markAllRead()`:
- Get unread IDs: `notifications.filter(n => !n.read).map(n => n.id)`
- If none, return.
- `supabase.from("notifications").update({ read: true }).in("id", unreadIds)`
- Update local state: map all to `read: true`
Per D-06: Triggered by "Mark all as read" text button.

`getTypeIcon(type: NotificationType)`:
Return JSX icon based on type (per research Pattern 7):
- `payment_success`: `<CreditCard className="w-4 h-4 text-green-400" />`
- `payment_failed`: `<CreditCard className="w-4 h-4 text-red-400" />`
- `task_created`: `<Plus className="w-4 h-4 text-blue-400" />`
- `task_updated`: `<CheckSquare className="w-4 h-4 text-brand-accent" />`

`formatTime(dateStr: string)`:
Use `formatDistanceToNow(new Date(dateStr), { addSuffix: true, locale: currentLocale === "es" ? es : undefined })` where currentLocale comes from `useLocale()`.

**JSX structure:**
```
<div ref={bellRef} className="relative">
  {/* Bell button */}
  <button onClick={() => setOpen(!open)} className="relative p-1.5 rounded-lg hover:bg-white/10 transition" aria-label={t("title")}>
    <Bell className="w-5 h-5 text-[#A3A39B]" />
    {unreadCount > 0 && (
      <span className={`absolute -top-1 -right-1 min-w-[18px] h-[18px] flex items-center justify-center rounded-full bg-brand-accent text-white text-[10px] font-bold px-1 ${pulse ? "animate-ping" : ""}`}>
        {badgeLabel}
      </span>
      {/* Static badge underneath ping animation */}
      <span className="absolute -top-1 -right-1 min-w-[18px] h-[18px] flex items-center justify-center rounded-full bg-brand-accent text-white text-[10px] font-bold px-1">
        {badgeLabel}
      </span>
    )}
  </button>

  {/* Dropdown */}
  {open && (
    <div className="absolute right-0 top-full mt-2 w-80 max-h-[400px] bg-brand-sidebar border border-white/10 rounded-lg shadow-xl z-50 flex flex-col">
      {/* Header */}
      <div className="flex items-center justify-between px-4 py-3 border-b border-white/10">
        <span className="text-white text-sm font-semibold">{t("title")}</span>
        {unreadCount > 0 && (
          <button onClick={markAllRead} className="text-brand-accent text-xs hover:underline">
            {t("markAllRead")}
          </button>
        )}
      </div>
      {/* List */}
      <div className="overflow-y-auto flex-1" style={{ maxHeight: "340px" }}>
        {notifications.length === 0 ? (
          <div className="flex flex-col items-center justify-center py-8 text-[#6B6B63]">
            <Bell className="w-8 h-8 mb-2 opacity-50" />
            <span className="text-xs">{t("empty")}</span>
          </div>
        ) : (
          notifications.map(n => (
            <button key={n.id} onClick={() => markRead(n.id)} className={`w-full flex items-start gap-3 px-4 py-3 text-left hover:bg-white/5 transition border-b border-white/5 ${!n.read ? "bg-white/[0.03]" : ""}`}>
              <div className="mt-0.5 flex-shrink-0">{getTypeIcon(n.type)}</div>
              <div className="flex-1 min-w-0">
                <div className="flex items-center justify-between gap-2">
                  <span className={`text-xs truncate ${!n.read ? "text-white font-semibold" : "text-[#A3A39B]"}`}>{n.title}</span>
                  <span className="text-[10px] text-[#6B6B63] flex-shrink-0">{formatTime(n.created_at)}</span>
                </div>
                <p className="text-[11px] text-[#6B6B63] truncate mt-0.5">{n.body}</p>
              </div>
              {!n.read && <div className="w-2 h-2 rounded-full bg-brand-accent mt-1.5 flex-shrink-0" />}
            </button>
          ))
        )}
      </div>
    </div>
  )}
</div>
```

Per D-04: Empty state has subtle Bell icon + "No notifications" text.
Per D-08: No toast, no sound -- badge increment only.
Per D-09: `animate-ping` class on badge span when `pulse` is true (2s duration then removed). Use the ping overlay pattern: one span with animate-ping on top, one static span below so the badge number is always readable.

**Tailwind classes:** Match portal sidebar theme -- bg-brand-sidebar, text-[#A3A39B], border-white/10, text-white for active items. These are the exact classes used in portal/layout.tsx.
