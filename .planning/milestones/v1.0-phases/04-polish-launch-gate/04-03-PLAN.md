  <action>
**Contrast audit methodology:**

1. Calculate contrast ratios for ALL text token + background token pairs using the WCAG formula. The key pairs to check:

**Light mode** (`:root`):
- `--color-text` (#1A1917) on `--color-bg` (#F8F7F4) — expected: ~16:1 (fine)
- `--color-text` (#1A1917) on `--color-surface` (#FDFCFA) — expected: ~17:1 (fine)
- `--color-text-muted` (#5C5C56) on `--color-bg` (#F8F7F4) — VERIFY: must be >= 4.5:1
- `--color-text-muted` (#5C5C56) on `--color-surface` (#FDFCFA) — VERIFY: must be >= 4.5:1
- `--color-text-subtle` (#6B6B64) on `--color-bg` (#F8F7F4) — annotated as 5.0:1 (verify)
- `--color-text-subtle` (#6B6B64) on `--color-surface` (#FDFCFA) — VERIFY
- `--color-accent` (#1B6B4A) on `--color-bg` (#F8F7F4) — for accent text uses
- `--color-accent` (#1B6B4A) on `--color-surface` (#FDFCFA) — for CTA button text

**Dark mode** (`@media (prefers-color-scheme: dark)`):
- `--color-text` (#F0EFE9) on `--color-bg` (#1A1917) — expected: high (fine)
- `--color-text` (#F0EFE9) on `--color-surface` (#242320) — expected: high (fine)
- `--color-text-muted` (#9C9B93) on `--color-bg` (#1A1917) — annotated 6.2:1 (verify)
- `--color-text-muted` (#9C9B93) on `--color-surface` (#242320) — VERIFY
- `--color-text-subtle` (#8C8B83) on `--color-bg` (#1A1917) — annotated 5.0:1 (verify)
- `--color-text-subtle` (#8C8B83) on `--color-surface` (#242320) — annotated 4.5:1 (verify closely)
- `--color-accent` (#259B6E) on `--color-bg` (#1A1917) — for dark mode accent text
- White text on `--color-accent` (#259B6E) — for CTA button text in dark mode

2. For any pair that fails 4.5:1:
   - Adjust the text token value in tokens.css (make it darker for light mode, lighter for dark mode)
   - Update BOTH `:root` (light) and `@media (prefers-color-scheme: dark) :root` (dark) blocks
   - Update the contrast ratio comment next to the token
   - Re-verify that the adjusted value still passes against ALL backgrounds it appears on

3. Also scan for any `color:` rules in base.css, landing.css, detail.css that use raw hex values instead of tokens — if found, convert to token references

**Important constraints:**
- Do NOT change background tokens (`--color-bg`, `--color-surface`) — only adjust text/foreground tokens
- Do NOT change `--color-accent` unless absolutely necessary (brand color)
- All changes must be in tokens.css per D-08
- Annotate every changed value with its new contrast ratio in a CSS comment
